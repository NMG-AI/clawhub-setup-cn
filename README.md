# clawhub-install-use

ClawHub 安装和使用指南技能（Skill），帮助用户快速上手 ClawHub 的安装、配置和技能管理。

## 功能

- ClawHub 安装与初始化
- 镜像配置（支持阿里云镜像）
- 技能的搜索、安装、更新、启用/禁用

## 安装

```bash
clawhub install clawhub-install-use
```

## 使用

安装后，当你在 Claude Code 中提到以下关键词时，该技能会自动激活：

- `clawhub`、`安装 clawhub`、`clawhub 怎么用`
- `安装技能`、`更新技能`、`管理技能`
- `clawhub init`、`clawhub install`、`clawhub update`
- `openclaw skills`
